# Hello World

A minimal Python project that prints "Hello, World!" to the console.

## Installation

Clone the repository:

```bash
git clone https://github.com/bedri-b/hello-world.git
cd hello_world
```

(Optional) Install as an editable package:

```bash
pip install -e .
```

## Usage

Run directly:

```bash
python hello_world.py
```

## Project Structure

```
hello_world/
├── hello_world.py   # main program
├── __main__.py      # module entry point
└── README.md        # this file
```
